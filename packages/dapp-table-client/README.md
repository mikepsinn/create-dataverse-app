# dapp-table-client

## Overview

Client for retrieving, creating and updating Dataverse dapps. Function list:

- getFileSystemModels
- getDapp
- getDapps
- createDapp
- updateDapp

## Install

```
pnpm install @dataverse/dapp-table-client
```
